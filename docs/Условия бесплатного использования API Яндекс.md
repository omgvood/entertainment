Условия бесплатного использования API Яндекс Карт
Для бесплатного использования продуктов API Яндекс Карт необходимо соблюдать ряд условий, которые перечислены ниже. Подробная информация об условиях бесплатного использования представлена по ссылке: https://yandex.ru/legal/maps_api/ru/.

Если вы нарушаете хотя бы одно из условий, рекомендуем оформить платную лицензию. Это поможет избежать блокировки доступа к API и сбоев в работе вашего сервиса.

1. API должен использоваться только в проектах с открытым доступом



Бесплатная версия API предназначена только для сайтов или приложений, доступ к которым может получить любой пользователь сети Интернет. Если есть регистрация, она должна быть доступной всем желающим без каких-либо ограничений в виде, например, дополнительной платы или регистрации, где требуется ввести данные удостоверения личности.

Также нельзя применять бесплатную версию API в тех случаях, когда решение, которое включает в себя использование API, разрабатывается с целью перепродажи.

2. Запрещается сохранять или изменять данные, полученные средствами API



Данные, полученные средствами API, нельзя сохранять. Например, нельзя один раз получить данные геокодера и затем сохранить их для дальнейшего использования в собственную базу/excel файл. Если необходимо сохранение данных – нужно приобрести расширенную коммерческую лицензию.

Данные, полученные через API, также нельзя изменять. Например, запрещено изменять ранжирование результатов поиска.

3. Запрещается скрывать и видоизменять копирайты и логотип Яндекса на карте, нужно обязательно выводить на экран кнопку «Открыть в Картах»



Копирайты, логотип Яндекса и кнопка перехода «Открыть в Картах» должны быть всегда видны на карте: их нельзя перекрывать другими элементами, скрывать или видоизменять. Это касается как интерактивных, так и статических карт. По этой причине запрещено размещение скриншотов карт. Для получения статического изображения карты вы можете воспользоваться Конструктором.

(Избежать соблюдения данного условия нельзя даже в коммерческой лицензии).

4. Нельзя использовать API для мониторинга и диспетчеризации



В бесплатной версии API запрещается использовать карту для отслеживания местоположения транспорта или мобильных сотрудников в реальном времени.

5. Нельзя превышать суточный лимит запросов к API



В случае регулярного превышения суточного лимита, доступ к API будет заблокирован без права восстановления. Подробная информация о лимитах представлена по ссылке:

https://yandex.ru/legal/maps_api/ru/.

6. Мобильное приложение должно содержать ссылку на Условия использования Яндекс Карт



В разделе приложения «О программе» (или другом похожем разделе) должна быть ссылка на Условия использования отдельных сервисов Яндекс Карт:

https://yandex.ru/legal/maps_api/
